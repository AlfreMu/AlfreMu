## About me

I build deployable software systems and the delivery pipelines behind them.

I'm a **DevOps & Cloud Engineer** from Argentina, focused on **CI/CD automation, Infrastructure as Code, containerization and cloud operations**.

I originally trained as a **lawyer**, which gave me a strong foundation in analytical thinking, documentation, communication and working with real deadlines. Today, I combine that background with hands-on technical work in `Linux`, `Docker`, `Kubernetes`, `GitHub Actions`, `Terraform`, `AWS` and observability tools.

I like building projects that are not just code that runs, but systems that can be **built, deployed, monitored and explained**.

- Spanish native | English B2
- Studying `University Programming Degree @ UTN`
- DevSecOps internship experience through `NOVIT & UTN`

- Building real products like [`PlayPoll`](https://github.com/AlfreMu/appvotarjuegos), a real-time voting app for choosing what to play with friends.
